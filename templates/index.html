🔬 Metodologia de Análise

📊

Análise por Plataforma

Cada rede social (Twitter/X, Instagram, Facebook) é analisada separadamente. Uma pessoa pode ser classificada diferente em cada rede — por exemplo, mais à esquerda no Twitter e centro no Facebook.

🔤

Palavras-chave

O modelo usa ~90 termos politicamente associados (45 esquerda, 45 direita). Cada ocorrência vale 1 ponto. Nomes de figuras políticas valem 2 pontos.

👥

Figuras Políticas

Banco de ~430 perfis classificados de -2 (esquerda forte) a +2 (direita forte). Seguir uma figura política vale 5× o score — é o indicador mais confiável.

⚖️

Imprensa = Centro (0)

Veículos como Folha, Estadão, G1, Globo e CNN Brasil recebem score 0 (neutro). Segui-los não indica posição política — são acompanhados por todos os espectros.

🐦

Twitter / X

Tweets públicos + quem a pessoa segue. Fonte mais rica: a análise de seguidores revela a "bolha" política.

📸

Instagram

Perfil público, bio e busca por menções. Sem API pública — depende de dados acessíveis publicamente na web.

👤

Facebook

Páginas públicas, publicações e curtidas. Sem API pública — usa busca web para encontrar conteúdo.

📰

Google News

Notícias recentes via RSS. Útil para figuras públicas que aparecem na mídia com frequência.

📖

Wikipédia

Artigos sobre figuras públicas. Identifica filiações partidárias e posicionamentos históricos.

🔍

DuckDuckGo

Busca web genérica para complementar quando as APIs diretas retornam poucos resultados.

📐 Escala de Classificação

-2 Esquerda
-1 Centro-Esq
0 Centro
+1 Centro-Dir
+2 Direita

A escala vai de -2 (esquerda forte) a +2 (direita forte). Veículos de imprensa tradicionais são mantidos em 0 (centro) para não distorcer a análise.

⚠️ Importante — Leia antes de usar

  • Este é um projeto de estudo acadêmico sobre análise de dados textuais e classificação.
  • A classificação é baseada em palavras-chave — não utiliza IA semântica ou processamento de linguagem natural avançado.
  • Apenas dados públicos são coletados — respeitando a LGPD e o Marco Civil da Internet.
  • Nenhum dado é armazenado — toda análise é processada em memória e descartada após exibição.
  • Instagram e Facebook não possuem APIs públicas — os melhores resultados são com Twitter/X.
  • O modelo não detecta ironia, sarcasmo ou citações contrárias — pode gerar falsos positivos.
  • Resultados com confiança BAIXA ou MUITO BAIXA devem ser interpretados com extrema cautela.
  • Consulte a documentação completa para detalhes sobre metodologia, calibração e aspectos legais.
